Kick

In this version, you are able to do reasoning, but if there's no reason, it will default to "None". Example: "||kick @b1nzy#0852 Cuz he's a ratelimiter"
If you want to ban, the bot requires the "KICK_MEMBERS" permission, and also, the moderator requires the "KICK_MEMBERS" permission.
If you want mod-logs, you require the "MANAGE_SERVER" permission, and the bot requires the "EMBED_LINKS" permission.
To set up mod-logs, type "||setting modlogs enabled true", and then type "||setting modlogs log-channel #mod-logs".
The default "modlogs log-channel" setting is "none" and the default for "modlogs enabled" is "false".
